# MarkDown 快速入门笔记

## 1. 最常用的代码块语法

### 例子
```html
<html>
    <head>
        <meta charset = "utf-8">
        <meta name="keywords" content="新手">
        <meta name="description" content="初学者制作的网页">
        <title>我的第一个网页</title>
        <link rel="shortcut icon" href="https://img.alicdn.com/tps/i3/T1OjaVFl4dXXa.JOZB-114-114.png" type="image/x-png">
    </head>
</html>
```
### 语法
\```内容\```
